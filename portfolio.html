Intro to GSAP
Quick Start for GSAP JavaScript Animations

GSAP is a JavaScript library for creating web animations. It is beginner-friendly, but also robust enough for professional web developers. I created this introduction to GSAP for novice web developers. It covers the basics of creating an animation and mentions a few deeper concepts like callback functions. Because GSAP can easily be used by novices, this guide cuts straight to the point to create simple animations. NiCE CXone: Getting Started
Guide for Developers Integrating with NiCE CXone

CXone is industry-leading contact center software (CCaaS). They have a thriving community of integrators, including both customers and partners. Integrators had minimal self-service resources and constantly asked Support about registering their app with NiCE and setting up the correct form of API authentication.

To solve this, I worked with a pair of lead engineers to create this getting started guide. Our approach included the following:

  • The guide needed to be instructional and task-oriented. This would help lead devs through the steps instead of obligating them to figure it out with minimal documentation.
  • The guide would work in tandem with an Authentication and Authorization Overview, which we created to be a more in-depth discussion of how API authentication worked with CXone.
  • The guide would prioritize a planning stage, which would also answer the FAQs that support engineers would handle.

The lead engineers informed me that this guide reduced their API-related support requests by 90%.

Voobly APIs
Voobly API Documentation Displayed with SwaggerUI

Voobly is a platform that lets you play PC games with friends or a competitive community. It supports peer-to-peer connections for a multiplayer experience. You can play casual games with friends, or competitively on ranked ladders.

The Voobly APIs have minimal public-facing documentation that is incomplete. This page displays the APIs with the Swagger UI and provides enhanced documentation. I made minor customizations to the Swagger styling to match the theme of the portfolio. Without working directly with the Voobly developers, I was also able to expand the information around how the APIs work and what information you can access.
